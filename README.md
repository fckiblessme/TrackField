# Мир легкой атлетики

Веб-приложение на фреймворке Bottle (Python), посвященное легкой атлетике. Сайт содержит информацию о различных дисциплинах, мировых рекордах, известных спортсменах и предстоящих соревнованиях. Проект выполнен в рамках лабораторной работы №2-4 по курсу МДК 02.02 
Инструментальные средства разработки программного обеспечения

## Начало работы

Эти инструкции помогут вам запустить копию проекта на вашем локальном компьютере для разработки и тестирования.

### Необходимые компоненты

Перед началом убедитесь, что у вас установлены следующие программы:

* **Python** (версия 3.8 или выше) - [Скачать Python](https://www.python.org/downloads/)
* **Git** (для клонирования репозитория) - [Скачать Git](https://git-scm.com/downloads)
* **Visual Studio 2022** (рекомендуется) или любой другой редактор кода (VS Code, PyCharm)

### Установка

Пошаговая инструкция по установке и запуску проекта:

**1. Клонируйте репозиторий**

Откройте терминал (командную строку) и выполните команду:
```bash
git clone https://github.com/fckiblessme/TrackField.git
```

**2. Перейдите в папку проекта**

```bash
cd TrackField
```

**3. Создайте виртуальное окружение**
```bash
# Для Windows
python -m venv env

# Для macOs/Linux
python3 -m venv env
```

**4. Активируйте виртуальное окружение**
```bash
# Для Windows
env\Scripts\activate

# Для macOS/Linux
source env/bin/activate
```

**5. Установите зависимости**
```bash
pip install -r requirements.txt
```

**6. Запустите приложение**
```bash
python app.py
```

**7. Откройте сайт в браузере**
Перейдите по адресу: http://localhost:8080

Вы должны увидеть главную страницу сайта с карточками всех дисциплин легкой атлетики

## Запуск тестов
